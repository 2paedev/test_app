# Test App

Ionic v6 test

## Features & Tools

- Ionic v6
- Angular 13
- Styles with SCSS
- Custom components
- Unit test with Karma/Jasmine (coverage)
- E2E testing with Cypress
- Lazy loading modules
- ESLint default configuration

## Getting Started

### Dependencies

- Angular-cli, Node, npm

### Installing

```bash
#!/bin/bash
npm install
```

### Executing

- Localhost server

```bash
#!/bin/bash
npm run start
```

- Unit test (no-watch)

```bash
#!/bin/bash
npm run test
```

- Unit test (watch)

```bash
#!/bin/bash
npm run test:watch
```

- Unit test (coverage)

```bash
#!/bin/bash
npm run test:coverage
```

- E2E test (Cypress Run)

```bash
#!/bin/bash
npm run e2e
```

- E2E test (Cypress Open)

```bash
#!/bin/bash
npm run e2e:open
```
